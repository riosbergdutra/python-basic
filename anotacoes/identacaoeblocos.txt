A indentação e os blocos de código são aspectos fundamentais em Python. Diferente de muitas outras linguagens de programação que usam chaves {} para definir blocos de código, Python usa a indentação para determinar o início e o fim de blocos de código. Vamos explorar como isso funciona.

Indentação em Python
A indentação em Python é feita utilizando espaços ou tabulações. O padrão geralmente aceito é usar 4 espaços para cada nível de indentação. Todos os códigos dentro de um bloco devem ser indentados com o mesmo número de espaços.

Exemplos de Indentação
Estruturas Condicionais com Indentação

x = 10

if x > 5:
    print("x é maior que 5")
    if x > 7:
        print("x também é maior que 7")
else:
    print("x não é maior que 5")
Explicação:

if x > 5: e o bloco associado são indentados com 4 espaços.
O segundo if dentro do primeiro if é indentado com mais 4 espaços, totalizando 8 espaços.
Estruturas de Repetição com Indentação

for i in range(5):
    print(f"Iteração {i}")
    if i % 2 == 0:
        print(f"{i} é par")
    else:
        print(f"{i} é ímpar")
Explicação:

O laço for é seguido por um bloco indentado com 4 espaços.
Dentro desse bloco, o if e else também são indentados com 4 espaços adicionais.
Blocos de Código
Um bloco de código é um grupo de declarações que são executadas juntas. Em Python, os blocos de código são definidos pela sua indentação.

def saudacao(nome):
    print(f"Olá, {nome}!")
    if nome == "Riosberg":
        print("Bem-vindo de volta, Riosberg!")

saudacao("Riosberg")
Explicação:

A definição da função def saudacao(nome): é seguida por um bloco de código indentado com 4 espaços.
Dentro desse bloco, há um if que também contém um bloco indentado com mais 4 espaços.
Importância da Indentação Correta
A indentação correta é crucial em Python, pois define a estrutura e o fluxo do programa. Erros de indentação podem levar a erros de sintaxe ou a comportamentos inesperados.

Exemplo de Erro de Indentação

x = 10

if x > 5:
print("x é maior que 5")  # Este código causará um erro de indentação
Erro:

O print("x é maior que 5") deve ser indentado com 4 espaços para estar dentro do bloco if.
