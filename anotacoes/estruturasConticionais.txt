As estruturas condicionais permitem que você execute diferentes blocos de código com base em condições específicas. Em Python, usamos as palavras-chave if, elif e else para construir essas estruturas. sendo assim a sintaxe:

if x > 5:
    print("x é maior que 5")
elif x == 5:
    print("x é igual a 5")
else:
    print("x é menor que 5")

elif é o else if do java
